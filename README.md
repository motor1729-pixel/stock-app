# 국내주식 전략 실험실

종목코드를 입력해 이동평균 전략을 과거 데이터로 시험하는 Streamlit 앱입니다.

## 기능

- KOSPI/KOSDAQ 종목의 일별 시세 조회
- 단기·장기 이동평균과 매수·매도 신호 표시
- 정확히 1주를 거래한다고 가정한 원화 손익 백테스트
- 누적수익률, 단순 보유 수익률, CAGR, MDD 비교
- KOSPI/KOSDAQ 대비 20일 상대수익률, RSI, 52주 가격 위치
- 종목별 최신 뉴스 제목, 출처, 게시 시각
- 전략과 1주 단순보유의 원화 평가액 차트

## 실행 방법

Python 3.11 이상에서 다음 명령을 실행합니다.

```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

브라우저가 열리면 종목코드와 시장을 선택하고 `백테스트 실행`을 누릅니다.

## 무료 배포

1. 이 폴더의 파일을 GitHub 저장소에 올립니다.
2. Streamlit Community Cloud에서 GitHub 저장소를 연결합니다.
3. 실행 파일로 `app.py`를 선택해 배포합니다.

## 주의사항

- 가격 데이터는 네이버 금융을 통해 가져오므로 지연·누락될 수 있습니다.
- 뉴스는 Google 뉴스 RSS를 통해 가져오며 제목만으로 호재·악재를 판정하지 않습니다.
- 신호는 당일 종가로 계산해 다음 거래일부터 적용합니다.
- 거래비용 입력값은 사용자가 실제 조건에 맞게 조정해야 합니다.
- 이 앱은 학습 및 연구용이며 투자 권유나 수익 보장 도구가 아닙니다.
